### Task: Creating a Deposit Rule:

1. Navigate to a specific plan and select Basic Deposit Rule

   ![image](https://github.com/OlegZas/SAP_Various/assets/115661636/076d00f4-eca8-492f-8e76-a3bb858ab586)

2. In the Output tab click Create New Deposit Amount

   ![image](https://github.com/OlegZas/SAP_Various/assets/115661636/12cdc156-2b27-4dd3-9986-15cbdb47fc62)

3. Open the editor window

   ![image](https://github.com/OlegZas/SAP_Various/assets/115661636/d8af0428-3256-48ab-b3c5-1eaeb91737a6)

4. Choose the Incentive which the rule will reference

   ![image](https://github.com/OlegZas/SAP_Various/assets/115661636/e65be523-970e-443b-a364-2a2b0fc98208)

5. Choose appropriate Earning Code and Earning Group

   ![image](https://github.com/OlegZas/SAP_Various/assets/115661636/fa3754e3-76b2-470f-98b6-130dc766677c)

6. Finish and Save.
